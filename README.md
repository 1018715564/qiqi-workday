# qiqi-workday
适用于企企经营管理平台的工时补全助手-在console中直接引入即可😁

## 使用方法

- 打开消息列表
- 打开控制台
- 输入下面代码

```js
import("https://cdn.jsdelivr.net/gh/1018715564/qiqi-workday@latest/index.js")
```

- 按照提示在7秒中之内滑动消息列表, 7秒之后自动输出结果
